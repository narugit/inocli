# inocli
Wrapper cli tool for [arduino/arduino-cli](https://github.com/arduino/arduino-cli)

## Usage
### Compile the sketch

```console
$ ./inocli.sh -c -f test.ino
```

### Upload the sketch

```console
$ ./inocli.sh -u -f test.ino
```

### Compile and upload the sketch

```console
$ ./inocli.sh -cu -f test.ino
```
